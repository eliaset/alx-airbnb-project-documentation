\# Flowchart - Property Booking Process



This flowchart visualizes the backend process of booking a property on the Airbnb Clone system.



\## Steps Covered

1\. Guest selects property and dates.

2\. System checks availability.

3\. Calculates total price.

4\. Processes payment through payment gateway.

5\. Creates booking record and updates property availability.

6\. Sends confirmation to guest and host.
